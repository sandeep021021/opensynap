# OpenSynap

🚧 Learning-in-public rewrite of OpenClaw (TypeScript/Node.js) into Python.

This is my journey to deeply understand:
- CLI runtimes and terminal UX
- Logging and observability
- Configuration and security boundaries
- Gateway routing and sessions
- Agent + provider architecture
- Channel adapters (Telegram/WhatsApp/Discord/etc.)
- Tool execution engines
- Production-grade system design

## Why?

Instead of only watching tutorials, I’m reverse engineering a real production open-source system
and rebuilding it from scratch in Python — one file at a time.
